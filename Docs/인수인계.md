## 중요 주의사항

### functions/.env 금지
`functions/.env`에 `KAKAO_REST_API_KEY`를 넣으면 Secret Manager와 충돌해 배포 실패.
- 에러: `Secret environment variable overlaps non secret environment variable`
- REST API key는 Firebase Secret Manager에만 존재해야 함
- `.env` 파일 자체를 만들면 안 됨
