# Moyeora Retention Strategy

## Engagement Triggers

홈에서 재방문 이유를 즉시 보이도록 3가지 카드형 트리거를 배치했습니다.

1. 이번 주 미응답자 수
2. 출석 상승 흐름(연속 참석)
3. 이번 달 활동 순위

또한 예정 일정 카드에서 사용자가 아직 응답하지 않았으면 `응답 필요` 배지를 표시해 즉시 행동을 유도합니다.

## Badge System (Simple)

프로필에 과도하지 않은 3개 배지만 노출합니다.

- `출석왕`: 출석 리더보드 Top 3
- `활동왕`: 활동 리더보드 Top 3
- `운영 기여왕`: owner/admin/treasurer 역할

배지는 동기 부여는 주되, 포인트/레벨/복잡한 보상 루프는 도입하지 않아 과몰입형 게이미피케이션을 피합니다.

## Avoid Over-Gamification

- 배지 수를 최소화(3개 고정)
- 순위 정보는 요약형으로만 표시
- 경쟁 요소보다 "해야 할 행동(응답)" 중심의 넛지 우선
- 연속 참여 정보도 가볍게 제공(강한 압박 문구 없음)

## Future Push Integration

향후 푸시 확장 방향:

1. 응답 미완료 일정만 정밀 알림
2. 개인 배지 달성 직후 요약 알림
3. 월말 활동 리포트 알림(요약 카드 링크)
4. 알림 피로도 방지를 위한 빈도 제한(주 단위 캡)
